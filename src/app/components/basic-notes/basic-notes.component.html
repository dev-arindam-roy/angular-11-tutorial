Basic Notes: Need to Know

Version:

- ~11.1.5

- ^11.1.5

- 11 is the major version | 1 is the minor version | 5 is the patch version

- major version means breaking changes

- minor version means new feature added

- patch version means bug fixed

- If add ~, means - update the patch version (if avaliable)

- If add ^, means - update the minor & patch version (if avaliable)

- If nothing add - need exact match during npm install or npm update

Angular:

- is javascript based frontend framework

- Developed and Maintaining by Google

- Built on TypeScript

- We can develop enterprise level application by Angular
